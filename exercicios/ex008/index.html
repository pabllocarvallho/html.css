Itálico / ÊNFASE

Nesta franse, temos um termo em itálico usando a tag I (não semântica)

Texto marcado

Podemos criar tambem um texto marcado usando a tag MARK

E no outro parágrafo, temos outro texto marcado no final.

Para que todas as marcações fiquem da mesma cor, podemos colocar no HEAD a tag style e dentro da tag mark background-color: e selecionar a cor desejada

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

tag Big e tag Small

Texto deletado

Podemos marcar um texto como excluido com a tag DELpara indicar que ele deve ser lido, mas nao considerado

Texto inserido

Podemos marcar um texto como inserido com a tag INS, para dar uma enfase e indicar que ele foi adicin

Existe também o sublinhado com a tag U (Não semântica)

Texto sobrescrito

Para inserir coisa do tipo, x20+3, com a tag SUP

Texto subescrito

Para inserir coisas do tipo H2O, com a tag SUB
